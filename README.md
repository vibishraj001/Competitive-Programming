# Competitive-Programming
Solving problems with best Time &amp; Space complexity within time limit

<h2>Programs Covered</h2>
#ProgramKey           Logic
1.Perfect Square  | sr * sr == n
2.Automorphic     | digit by digit % 10 comparison
3.Harshad         | Numbersum of digits divides n
4.Abundant Number | sum of proper divisors > n

<h2>Golden Rules</h2>

1.Protect original variable → use temp
2.Never divide by zero → start from i = 1
3.Output must be exact → no extra text
4.return false = return 0, return true = return 1
5.bool is cleaner than int for true/false
