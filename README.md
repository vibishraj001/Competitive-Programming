# Competitive-Programming
Solving problems with best Time &amp; Space complexity within time limit

<h2>Programs Covered</h2>
#ProgramKey           Logic<br>
1.Perfect Square  | sr * sr == n<br>
2.Automorphic     | digit by digit % 10 comparison<br>
3.Harshad         | Numbersum of digits divides n<br>
4.Abundant Number | sum of proper divisors > n<br>

<h2>Golden Rules</h2>

1.Protect original variable → use temp<br>
2.Never divide by zero → start from i = 1<br>
3.Output must be exact → no extra text<br>
4.return false = return 0, return true = return 1<br>
5.bool is cleaner than int for true/false<br>
